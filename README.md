# HOSPITAL PERFORMANCE ANALYSIS DASHBOARD
## Project Overview
This project analyzes hospital patient records to identify trends in diagnosis, patient demographics, doctor workload, and treatment costs.

## Tools Used
This project was completed using:
-	Microsoft Excel for initial data preparation
-	MySQL for SQL analysis
-	Power BI for dashboard creation and DAX analysis

## Data Cleaaning & Preparation
Data preparation involved:
-	checking missing values, 
-	formatting dates, 
-	categorizing age groups, 
-	validating doctor assignments, 
-	and standardizing diagnosis categories.

## Analysis Performed
The analysis included:
- Diagnosis analysis
- Doctor workload analysis
- Patient demographic analysis
- Monthly trend analysis
-	Treatment cost analysis, 
- dynamic filtering using slicers.

## Key Insights
Key insights from the analysis include:
- Malaria and hypertension were the most common diagnoses, 
-	adult patients represented the majority of hospital visits, 
-	doctor workload distribution appeared uneven, 
-	and male patients accounted for a slightly larger share of hospital visits. 

## Recommendations 
Recommendations include:
-	reviewing doctor workload allocation, 
-	prioritizing resources toward common diagnoses, 
-	monitoring patient demographic trends, 
-	and conducting further investigation into healthcare utilization patterns across age groups.

## Technical Skills Demonstrated
Skills demonstrated in this project:
-	SQL querying and aggregation, 
-	DAX calculations, 
-	data modeling, 
-	Power BI dashboard design, 
- and analytical storytelling.
